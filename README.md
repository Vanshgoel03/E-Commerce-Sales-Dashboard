# E-Commerce-Sales-Dashboard
An interactive Power BI dashboard designed to visualize and analyze e-commerce sales data, tracking key metrics such as revenue, order volume, average order value (AOV), and product performance.
Power BI E-commerce Sales Analytics Dashboard
Developed an interactive Power BI dashboard to analyze e-commerce sales data, delivering insights into revenue growth, order trends, average order value (AOV), and top-performing products. The dashboard leverages dynamic slicers, KPIs, and visual storytelling to support business and marketing decisions.
